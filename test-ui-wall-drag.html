Wall Dragging UI Test Instructions

  1. Open the editor at http://localhost:5173/editor/5
  2. Switch to Select/Pointer tool (first toolbar button)
  3. Click on the room in the 3D viewport
  4. Verify blue edge handles appear on all four walls:
    • Front edge (toward +Z)
    • Back edge (toward -Z)
    • Right edge (toward +X)
    • Left edge (toward -X)
  5. Drag an edge handle outward or inward
  6. Verify dimension labels update in real-time
  7. Release the mouse button
  8. Verify a success toast appears: "Room resized"
  9. Verify the new dimensions are displayed
  10. Refresh the page to confirm dimensions persisted

Expected Behavior

  • Edge handles only visible when room is selected
  • Edge handles are semi-transparent blue boxes along each wall
  • Minimum room size is 1m × 1m (cannot drag smaller)
  • Dimension labels show current size during drag
  • Toast notification confirms successful resize
  • Changes persist to database immediately
